# Vanilla Capital Backoffice

Client information and compliance management platform for Vanilla Capital.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- **Main Page:** Placeholder with branding image
- **Clients:** CRM-style list with sort, search, copy-to-clipboard, PDF preview. Profile view with one-pager for all client attributes including marital info.
- **Compliance:** Document library (Onboarding & Corporate) with PDF upload/viewer. KYC and Suitability Form placeholders.

## Branding

- Colors: Main `#1A2433`, Secondary `#C8B991`, Black `#1A1A1A`, White `#F8F6F0`
- Fonts: Canela family (from `/public/fonts`)
- Dark/light mode toggle
