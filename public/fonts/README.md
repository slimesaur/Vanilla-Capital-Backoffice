# Fonts

Add the following font files to this directory:

## Canela (Titles)
- `Canela-Regular-Trial.otf`
- `Canela-Bold-Trial.otf`
- `CanelaText-Regular-Trial.otf`
- `CanelaText-Bold-Trial.otf`
- `CanelaDeck-Regular-Trial.otf`
- `CanelaDeck-Bold-Trial.otf`

Source: Commercial Type

## Arpona (Subtitles / Third titles)
- `Arpona-Regular.woff2` or `Arpona-Regular.woff`

Source: BeFonts, Font Canyon, or MyFonts (commercial license)

## Inter Tight (Body text)
Loaded via `@fontsource/inter-tight` npm package – no manual installation needed.
