# CVM Compliance Documents

This directory should contain the following mandatory PDF documents according to CVM regulations:

## Required Documents

1. **compliance.pdf** - Compliance Policy
   - Document that establishes Vanilla Capital's compliance guidelines and procedures

2. **ethics.pdf** - Code of Ethics and Conduct
   - Code that defines the ethical principles and expected conduct standards

3. **trading.pdf** - Personal Trading and Investment Policy
   - Policy that establishes rules for employee personal trading and investments

4. **reference.pdf** - Reference Form
   - Reference form with company information, services and contact details

## How to Add

1. Place the PDF files in this directory (`public/documents/`)
2. Make sure file names match exactly as listed above
3. Documents will be automatically available for download on the `/compliance` page

## Note

Currently, the download links are configured to look for these files. When the real documents are ready, simply replace the placeholders or add the files with the correct names.
